# libfvad: voice activity detection (VAD) library
Most of the code is from [https://github.com/reedom/libfvad_ios].

Swift wrapper.

## Installation

libfvad_ios is available swift package. 

```
dependencies: [
    .package(url: "https://github.com/gfreezy/libfvad")
]
```
